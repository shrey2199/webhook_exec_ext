Your Webhooks

Available Variables

Use these variables in your webhooks using the {var_name} syntax

{page_url} Full page URL (or clicked link URL)
{page_title} Page title
{page_domain} Page domain
{page_protocol} Protocol (http/https)
{selected_text} Currently selected text on page
{timestamp} Unix timestamp (milliseconds)
{date} Current date (YYYY-MM-DD)
{time} Current time (HH:MM:SS)
{datetime} ISO 8601 datetime
